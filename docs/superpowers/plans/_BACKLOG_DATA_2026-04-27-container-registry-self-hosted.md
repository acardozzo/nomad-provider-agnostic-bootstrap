# Container Registry (Self-Hosted) Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Run a private container registry (Docker Distribution) as a Nomad job behind Traefik with TLS and basic-auth, plus blob storage on MinIO. Closes audit #14.

**Architecture:** `registry:2.8` Nomad service job, blobs stored in MinIO via the s3 storage driver, fronted by Traefik with basic-auth middleware (separate htpasswd from dashboard).

**Tech Stack:** registry 2.8.x, MinIO (from object-storage plan), Traefik, htpasswd from secrets playbook.

## Task 1: Defaults + smoke

```bash
cat >> ansible/inventory/group_vars/all/defaults.yml <<'EOF'

# Container registry
registry_version: "2.8.3"
registry_host: "registry.{{ traefik_domain }}"

cat >> ansible/inventory/group_vars/all/secrets.example.yml <<'EOF'
registry_user: "orbty"
registry_password: ""              # openssl rand -base64 32

cat > tests/smoke/test_registry.sh <<'EOF'
#!/usr/bin/env bash
set -euo pipefail
if [[ $# -lt 1 ]]; then echo "usage: $0 <vm>" >&2; exit 64; fi
VM="$1"
USER=$(awk '$1=="registry_user:" {print $2}' "$(dirname "${BASH_SOURCE[0]}")/../../ansible/inventory/group_vars/all/secrets.yml" | tr -d \")
PASS=$(awk '$1=="registry_password:" {print $2}' "$(dirname "${BASH_SOURCE[0]}")/../../ansible/inventory/group_vars/all/secrets.yml" | tr -d \")

echo "=== Registry catalog endpoint reachable ==="
multipass exec "$VM" -- curl -sk -u "$USER:$PASS" https://registry.cluster.local/v2/_catalog | grep -q repositories || { echo FAIL; exit 1; }
echo OK

echo "=== docker login + push + pull ==="
multipass exec "$VM" -- bash -c "
  echo '$PASS' | sudo docker login registry.cluster.local -u '$USER' --password-stdin
  sudo docker pull alpine:3.20
  sudo docker tag alpine:3.20 registry.cluster.local/smoke/alpine:3.20
  sudo docker push registry.cluster.local/smoke/alpine:3.20
  sudo docker rmi registry.cluster.local/smoke/alpine:3.20
  sudo docker pull registry.cluster.local/smoke/alpine:3.20
"
echo OK

echo "ALL REGISTRY CHECKS PASSED"
EOF
chmod +x tests/smoke/test_registry.sh

git add ansible/inventory/group_vars/all/defaults.yml ansible/inventory/group_vars/all/secrets.example.yml tests/smoke/test_registry.sh
git commit -m "test(registry): defaults + failing smoke"
```

## Task 2: Secrets playbook extension

Append to `ansible/playbooks/secrets.yml`:

```yaml
    - name: Generate registry password (if missing)
      ansible.builtin.shell: |
        set -o pipefail
        openssl rand -base64 32 | tr -d '\n'
      args: { executable: /bin/bash }
      register: registry_pw_gen
      when: existing_secrets.registry_password | default('') == ''
      changed_when: existing_secrets.registry_password | default('') == ''

    - name: Compute registry htpasswd
      ansible.builtin.command:
        argv: [htpasswd, -nbB, "{{ registry_user | default('orbty') }}", "{{ existing_secrets.registry_password | default(registry_pw_gen.stdout | default('')) }}"]
      register: registry_htpasswd_gen
      changed_when: false

    - name: Merge registry secrets
      ansible.builtin.set_fact:
        merged_secrets: "{{ merged_secrets | combine({
          'registry_user': existing_secrets.registry_user | default('orbty'),
          'registry_password': existing_secrets.registry_password | default(registry_pw_gen.stdout | default('')),
          'registry_htpasswd': registry_htpasswd_gen.stdout
        }) }}"
```

```bash

git commit -m "feat(secrets): generate registry password + htpasswd"

## Task 3: Registry config + job

```bash
mkdir -p ansible/roles/registry/{tasks,templates}
cat > ansible/roles/registry/templates/config.yml.j2 <<'EOF'
version: 0.1
log:
  level: info
storage:
  s3:
    accesskey: "{{ minio_root_user }}"
    secretkey: "{{ minio_root_password }}"
    region: us-east-1
    regionendpoint: http://minio.service.consul:9000
    bucket: "{{ registry_bucket }}"
    secure: false
    v4auth: true
  cache:
    blobdescriptor: inmemory
http:
  addr: :5000
  headers:
    X-Content-Type-Options: [nosniff]
auth:
  htpasswd:
    realm: orbty-registry
    path: /local/htpasswd
EOF

cat > ansible/roles/registry/templates/registry.nomad.hcl.j2 <<'EOF'
job "registry" {
  datacenters = ["{{ nomad_datacenter }}"]
  type        = "service"

  group "registry" {
    count = 1

    constraint { attribute = "${node.class}" operator = "=" value = "server" }

    network {
      mode = "host"
      port "http" { static = 5000 }
    }

    service {
      name = "registry"
      port = "http"
      check { type = "tcp" port = "http" interval = "10s" timeout = "2s" }
      tags = [
        "traefik.enable=true",
        "traefik.http.routers.registry.rule=Host(`{{ registry_host }}`)",
        "traefik.http.routers.registry.tls=true",
{% if acme_enabled %}
        "traefik.http.routers.registry.tls.certresolver=le",
{% endif %}
      ]
    }

    task "registry" {
      driver = "docker"

      template {
        data = <<EOT
{{ '{{' }} key "registry/config.yml" {{ '}}' }}
EOT
        destination = "local/config.yml"
        change_mode = "restart"
      }

      template {
        data = "{{ registry_htpasswd }}\n"
        destination = "local/htpasswd"
        perms = "0644"
      }

      config {
        image        = "registry:{{ registry_version }}"
        network_mode = "host"
        args = ["serve", "/local/config.yml"]
      }

      resources { cpu = 200; memory = 256 }
    }
  }
}
EOF

cat > ansible/roles/registry/tasks/main.yml <<'EOF'
---
- name: Render registry config to KV
  ansible.builtin.uri:
    url: "http://{{ hostvars[groups['servers'][0]].ansible_host }}:8500/v1/kv/registry/config.yml"
    method: PUT
    body: "{{ lookup('template', 'config.yml.j2') }}"
    headers: { X-Consul-Token: "{{ consul_bootstrap_token }}" }
    status_code: 200
  delegate_to: localhost
  become: false
  run_once: true

- name: Submit registry
  ansible.builtin.shell: nomad job run -
  args:
    stdin: "{{ lookup('template', 'registry.nomad.hcl.j2') }}"
    executable: /bin/bash
  environment:
    NOMAD_ADDR: "http://{{ hostvars[groups['servers'][0]].ansible_host }}:4646"
    NOMAD_TOKEN: "{{ nomad_bootstrap_token }}"
  delegate_to: localhost
  become: false
  run_once: true
  changed_when: true

- name: Pre-create bucket in MinIO
  ansible.builtin.shell: |
    mc alias set orbty http://minio.service.consul:9000 {{ minio_root_user }} {{ minio_root_password }} 2>/dev/null
    mc mb -p orbty/{{ registry_bucket }} || true
  delegate_to: "{{ groups['servers'][0] }}"
  changed_when: true
EOF

git add ansible/roles/registry/
git commit -m "feat(registry): registry:2 nomad job with minio s3 backend"
```

# Runbook — Container Registry

## Endpoint
`https://{{ registry_host }}` (basic-auth, htpasswd from secrets.yml).

## Storage
Blobs live in MinIO bucket `{{ registry_bucket }}`. To switch to off-cluster (B2 / Vultr Object Storage), update `regionendpoint` in `config.yml.j2`.

## docker login from a workstation
```bash
docker login registry.<domain>
# user: orbty
# password: <registry_password from secrets.yml>
```

## Push your first image
```bash
docker tag myapp:latest registry.<domain>/myteam/myapp:1.0.0
docker push registry.<domain>/myteam/myapp:1.0.0
```

## Garbage collection
Run periodically (manually or as a Nomad batch job):
```bash
docker exec -it $(docker ps --filter ancestor=registry:2.8.3 -q) \
  registry garbage-collect /local/config.yml
```
